=====================================================
リモートの変更をローカルにマージ
①マージ先のブランチにチェックアウト
git checkout feature/STK-0001

②リモートリポジトリからローカルのリモート追跡リポジトリに変更をダウンロード
git fetch

③ローカルのリモート追跡リポジトリからローカルのブランチにマージ(ダウンロード)
git merge origin/feature/STK-0001

=====================================================
feature/STK-0001の変更をローカルのdevelopにマージして、リモートリポジトリpush
①マージ先のブランチにチェックアウト
git checkout develop

②ローカルのfeature/STK-0001ブランチの変更内容をローカルのdevlopにマージ
git merge feature/STK-0001

git logで見ると、feature/STK-0001ブランチの変更内容がローカルのdevelopに反映されている。
ここでは、リモートdevelopは変わらない。

③ローカルのdevelopを、リモートのdevelopにアップロード(push)
git push origin develop
